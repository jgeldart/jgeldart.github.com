Selected Projects

Drove the development of a semantic HTML view system capable of being themed entirely using CSS. This system made Plone the world’s first WAI-AAA compliant content management system. Development work used XHTML, CSS, Javascript and Python.

Mercury Web-Service Matchmaking System

Designed and built a matchmaking system for semantic web-services based on natural language descriptions as part of my PhD project. Used Prolog, Python and Java for various components of the system. Service descriptions were given in WSDL and OWL-S.

Designed and built a shared information layer for the GNOME desktop environment for my bachelors dissertation using Python.
